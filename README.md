# Ejercicio Práctico - Módulo 5: Finanzas Personales

Este proyecto contiene las consultas SQL desarrolladas para resolver el ejercicio práctico del Módulo 5 del curso Fullstack Talento Digital.

## Descripción
Se trabajó con una tabla llamada `finanzas_personales` que almacena información sobre deudas y créditos personales. A partir de los datos entregados, se resolvieron distintas consultas para responder preguntas sobre el estado financiero, tales como:
- ¿A quién le debo más dinero?
- ¿Quién me debe más dinero?
- ¿Cuánto debo en total y en promedio?
- ¿Cuántos meses me tomaría pagar todo?
- ¿Cómo cambia mi deuda si cobro lo que me deben?
- ¿Cuánto debo pagar mensualmente?
- Inserción y actualización de registros según nuevos eventos.

## Archivos
- `ejercicio_3_mod_5_querys.sql`: Contiene la creación de la tabla, inserción de datos y todas las consultas SQL comentadas según los requerimientos del ejercicio.

## Instrucciones de uso
1. Abre el archivo `.sql` en tu gestor de base de datos (por ejemplo, pgAdmin).
2. Ejecuta el script completo para crear la tabla y cargar los datos.
3. Ejecuta cada consulta para responder a las preguntas del ejercicio.

## Notas
- Cada consulta está comentada con la pregunta correspondiente del enunciado.
- Puedes modificar los datos o agregar nuevas consultas según lo requiera la actividad.
